# Demo Task List

This folder was created by following a Cursor task list, step by step.

## What is this?

A minimal example showing how Cursor manages multi-step tasks using its built-in todo system: tasks are created, executed one by one, and marked complete as they finish.

## Completed steps

1. **Create a demo folder** — `demo-task-list/` was created.
2. **Add README.md** — This file with project description.
3. **Add config** — `demo-config.json` with name, version, and description.
4. **Summary** — This section listing all completed steps.
