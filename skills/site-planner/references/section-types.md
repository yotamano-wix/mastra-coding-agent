# Section Type Catalog

## Header
- **Purpose**: Site navigation and branding
- **Mandatory**: Logo component (100px width), Menu component
- **Optional**: Max 1 CTA button
- **Prohibited**: No other components, no text boxes for site name, no container boxes
- **Structure**: Single column only
- **Layout options**: Centered | Right-Aligned | Left-Aligned
- **Styles**: Standard (full-width solid bg) | Compact (minimal height) | Compact + Promotional strip | Bordered (bottom border only)
- **Visual**: Background from Base 1, Base 2, Shade 1-2, or Accent 1-2 only. No lines. No separate dividers.
- **Logo**: ALWAYS positioned left, width exactly 100px (w-25), 30px padding to right

## Hero (Home Page Only)
- **Purpose**: First impression, tone-setting, statement headline
- **Must include**: Animation, visuals, impactful typography, image
- **Typography**: Use H1 for title
- **Text alignment**: Centered or left only (NEVER right)
- **Restriction**: If full-section background image â†’ text MUST be in a container with solid bg for accessibility
- **Key**: Create standout layout that defies expectations

## About
- **Purpose**: Short overview of business story
- **Requirements**: MUST include an image

## List / Services / Features
- **Purpose**: Showcase 3-6 items (services, projects, features, etc.)
- **Content**: Section header + items (title, optional subtitle, text, optional image, optional button)
- **Typography**: Use H5 for item titles
- **Card rules**: Equal-size containers, 2-4 cards per row, 5-10% section width gap
- **Alignment**: All text same alignment, images same size, buttons docked to bottom
- **Card height**: Max 40% of section height

## Testimonials
- **Purpose**: Client reviews
- **Content**: Quotes, name, optional image, optional star rating
- **Typography**: Testimonial text no larger than H6

## Promotional
- **Purpose**: Special offers, announcements, conversion CTAs
- **Content**: CTA buttons, promotional text, image
- **Can include**: Text marquee component

## Contact
- **Purpose**: Clear way to reach out
- **Content**: Contact details (Text + Button components). Note: ContactForm component does NOT exist in Wix

## Core Functionality (Wix App Widgets)
- **Purpose**: Display native Wix app widget matching site functionality
- **Widget selection**: Productsâ†’Slider Product Gallery | Blogâ†’Post List Widget | Eventsâ†’Event Widget | Servicesâ†’Service List Widget
- **Rule**: Only include if section description explicitly calls for it

## Footer
- **Purpose**: Business info and secondary navigation
- **Background**: MUST match header luminance group
- **Content**: Business name, logo (optional, 100px width), contact info, social media links, copyright
- **Prohibited**: No navigational links
- **Year**: Use 2026

## Forbidden Layouts
- Horizontal scroll
- Carousel
- Slideshow
- Sticky sections
- Accordion
