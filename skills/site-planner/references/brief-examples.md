# Example Local Briefs

## Example A — Hero: Two-Module Gallery

```
- [ ] **Hero** (section-id: hero-1) — Bold visual introduction with dual imagery
  - Layout: Full-width, Base 1, 40px edge spacing
    Top: Two equal-width images side-by-side, 20px gap, 350px height
    Bottom: Two cells aligned with images. Left: Headline (H2). Right: Description (P2)
    Footer bar: Email left, CTA right
  - Background: Base 1
  - Spacing: 40px below labels to content
  - Key: Perfect vertical/horizontal alignment, editorial aesthetic
```

## Example B — Hero: Asymmetric Split

```
- [ ] **Hero** (section-id: hero-1) — Dynamic split layout with dominant imagery
  - Layout: Full-width, Base 1
    Left 45%: text + small image stacked
    Right 55%: full-height hero image
  - Content: Headline "Freshly Baked Goods" (H1, Base 2), Subhead (P1), CTA secondary button
  - Background: Base 1
  - Spacing: 60px headline→subhead, 80px CTA→small image
  - Key: Headline dominates top-left, right image bleeds edges, small image anchors bottom-left
```

## Example C — List Section: Services Cards

```
- [ ] **Our Services** (section-id: services-1) — Showcase core business offerings
  - Layout: Section header top, 3 equal cards in row below
    Each card: Image top (1:1, 200px), Title (H5), Description (P2), CTA button bottom
  - Background: Shade 1
  - Spacing: 24px card gap, 40px section padding
  - Key: Equal card heights, buttons docked bottom, images docked top, consistent text alignment left
```

## Example D — Footer: Minimal Two-Column

```
- [ ] **Footer** (section-id: footer-1)
  - Layout: Two columns. Left: Logo + tagline (P3). Right: Contact info stack + social links horizontal
  - Background: Base 2 (matches header luminance)
  - Spacing: 60px top/bottom, 40px sides
  - Key: Logo 100px width, text Base 1, social bar horizontal
```
